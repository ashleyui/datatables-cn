扩展(Extensions)

Datatables提供的特性可以极大地增强了这个页面上可用的插件的使用,提供给用户交互和配置选项

  • 自动填充(AutoFill)
  • 按钮(Buttons)
  • 列排列(ColReorder)
  • 编辑器(Editor)
  • 列固定(FixedColumns)
  • 表头固定(FixedHeader)
  • KeyTable
  • 自适应(Responsive)
  • 行排列(RowReorder)
  • 滚动渲染器(Scroller)
  • 选择(Select)
  • 列隐藏显示(ColVis)（废弃）
  • 表格工具按钮(TableTools)（废弃）
{% include xad.html %}

Datatables提供了一些扩展，使表格变的更强大，使用下面的扩展可以很容易实现更复杂的功能，看看下面有哪些高级功能

自动填充(AutoFill)

自动填充添加了类似Excel那样，通过拖拽覆盖到的单元格被填充或者自增的功能 更多 »

按钮(Buttons)

Datatables的按钮扩展提供了一组常用的选项。包含了导出，打印和列的显示控制。 更多 »

列排列(ColReorder)

ColReorder 允许用户通过下拉或者拖拽列头修改列的排列顺序 更多 »

编辑器(Editor)

Editor 给你的表格提供一个完整的编辑能力，包括多行很行内编辑。使用Editor 创建一个完全自定义的表格，让用户修改表格中的信息，这个并不简单。 更多»

列固定(FixedColumns)

FixedColumns 提供列冻结的功能，比如用户始终能看到索引列，而右边的表格通过滚动条横向拖动。 更多»

表头固定(FixedHeader)

FixedHeader 提供冻结header，footer和大多数左边或者右边的列，确保标题信息永远都是可见的。 更多»

KeyTable

KeyTable 提供类似Excel 单元格导航。事件（获得焦点 focus，失去焦点 blur，动作 action 等等）可以分配指定到 单元格、列、行或者所有的单元格中。 更多»

自适应(Responsive)

Responsive 将提供自动适应屏幕的功能，让你的表格在不同的设备上都能够显示。 更多»

行排列(RowReorder)

RowReorder 提供给用户能够重新排列行的功能（点击拖动/鼠标接触拖动）。与Editor集成，可以多行编辑后直接保存更新后的数据。 更多»

滚动渲染器(Scroller)

scroller是一个虚拟的渲染器。允许表格看起来是有滚动条在拖动整个数据，实际上只有显示的部分的数据被拖出来，可以更快的操作。 更多»

选择(Select)

Select 提供选择行的能力，被选中的可以是行，列，或者单元格，可以是一起的也可以是独立的。这些被选中的，都可以使用api来直接操作。 更多»

列隐藏显示(ColVis)废弃

ColVis 已经废弃了，请使用Buttons扩展。ColVis 允许用户操作列的隐藏和显示。 更多»

表格工具按钮(TableTools)废弃

TableTools 已经废弃了，请使用Buttons扩展和Select扩展。TableTools 功能提供 打印，保存文件，复制到剪贴板。还有行的选择能力。 更多»

ColVis 和Buttons 已经废弃了，取而代之的是Buttons和Select扩展。强烈建议不再使用这两个废弃的扩展。
