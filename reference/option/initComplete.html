Datatables初始化完毕(initComplete)

起始版本号: {{ page.since }}

当表格完成加载绘制完成后执行此方法。

说明

表格初始化是在ajax之前，所以这个回调是指在所有数据都加载完成后

数据类型

函数（function）initComplete( settings, json )

参数:
名称 类型 是否可选
1 settings

{% include href/type/DataTables.html param="DataTables.Settings" %}

No

datatables的设置对象
2 json

{% include href/object.js %}

No

如果使用ajax选项来获取数据，则得到服务器返回的数据，否则是 undefined

示例

表格加载完成弹出提示框 

$('#example').dataTable( {
  "initComplete": function(settings, json) {
    alert( 'DataTables has finished its initialisation.' );
  }
} );

当表格加载时，显示[正在加载……]的的字样 (设置 {% include href/option/Features.html param="processing" %} 属性也可以达到这个效果) 

$('<div class="loading">正在加载……</div>').appendTo('body);

$('#example').dataTable( {
  "initComplete": function( settings, json ) {
    $('div.loading').remove();
  }
} );
