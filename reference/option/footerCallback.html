tfoot回调(footerCallback)

起始版本号: {{ page.since }}

tfoot显示回调方法

说明

同{% include href/option/Callbacks.html param="headerCallback" %}回调一样，只不过这个是用来处理tfoot的每一次重绘事件

注意：如果没有tfoot，此回调函数将不会被执行

数据类型（Type）

function footerCallback( tfoot, data, start, end, display )

参数:
名称 类型 是否可选
1 tfoot {% include href/node.js %} No

tfoot就是table的表脚

2 data {% include href/type/Javascript.html param="array" %} No

table的所有数据对象

3 start {% include href/integer.js %} No

Index for the current display starting point in the display array

4 end {% include href/integer.js %} No

Index for the current display ending point in the display array

5 display {% include href/type/Javascript.html param="array" %} No

Index array to translate the visual position to the full data array

示例

修改tfoot里的内容 

$('#example').dataTable( {
  "footerCallback": function( tfoot, data, start, end, display ) {
    $(tfoot).find('th').eq(0).html( "Starting index is "+start );
  }
} );
{% include codedebug.html param='http://www.gbtags.com/gb/rtwidget-replayerpreview/690.htm' %}

使用reduce()方法实现合计功能 

$('#example').dataTable( {
  "footerCallback": function( tfoot, data, start, end, display ) {
    var api = this.api();
    $( api.column( 5 ).footer() ).html(
        api.column( 5 ).data().reduce( function ( a, b ) {
            return a + b;
        } )
    );
  }
} );
