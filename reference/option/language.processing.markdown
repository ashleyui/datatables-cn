## 描述(Description)

当表格正在执行用户操作的时候显示的字符串(通常是一个排序操作或者类似的)


## 类型(Type)
这个选项能够接受以下类型的参数：

- {% include href/Types.html param="string" %}


## 默认值(Default)
- Value ：`Processing...`

 
## 例子(Example)

设置处理过程中显示的文字
{% highlight javascript linenos %}
$('#example').DataTable( {
    "language": {
        "paginate": {
          "processing": "正在处理中。。。"
        }
      }
} );
{% endhighlight %}
