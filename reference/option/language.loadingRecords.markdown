## 描述(Description)

当DataTalbes使用ajax异步获取数据和第一次渲染时，该消息显示在表格的空行中，以向最终用户表示数据正在加载中。

需要注意，此参数只在客户端处理模式下有效，在服务器模式下不使用此参数。

## 类型(Type)
这个选项能够接受以下类型的参数：

- {% include href/Types.html param="string" %}

## 默认值(Default)
- Value ：`Loading...`

 
## 例子(Example)

设置加载数据的提示字符串
{% highlight javascript linenos %}
$('#example').DataTable( {
 "ajax": "json.txt",
  "language": {
       "loadingRecords": "请等待，数据正在加载中......"
    }
} );
{% endhighlight %}
