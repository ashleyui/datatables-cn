## 描述(Description)

此字符串告诉最终用户表格没有任何记录。通常此字符串的格式应该和`info` 匹配。


## 类型(Type)
这个选项能够接受以下类型的参数：

- {% include href/Types.html param="string" %}

## 默认值(Default)
- Value ：`Showing 0 to 0 of 0 entries`

 
## 例子(Example)

没有记录可以显示
{% highlight javascript linenos %}
$('#example').DataTable( {
   "language": {
       "infoEmpty": "没有记录可以显示"
     }
} );
{% endhighlight %}
