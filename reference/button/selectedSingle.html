selectedSingle

起始版本号: {{ page.since }}

一个项目被选中时需要开启

注意：这个需要使用 Select扩展的支持

说明(Description)

这个按钮和 {% include href/button/selected.button %} 有些类似，不同的是这个针对的只是单个的选项。

单项选择是很有用的，当你希望执行很复杂的操作，比如编辑一行中的数据。

通过一个 action的 function提供此功能， 参考 则这个按钮会被激活，参考 {% include href/option/buttons.buttons.action %}/{% include href/api/Buttons.html param="button().action()" %}

示例

Datatables初始化：在控制台中打印当前选择的数据 

$('#myTable').DataTable( {
    buttons: [
        {
            extend: 'selectedSingle',
            text: 'Log selected data'
            action: function ( e, dt, button, config ) {
                console.log( dt.row( { selected: true } ).data() );
            }
        }
    }
} );
