columnToggle

起始版本号: {{ page.since }}

一个按钮,切换一个或多个列的可见性

注意：这个需要使用Bottons扩展

说明(Description)

如果你不希望给每个按钮都配置可以切换显示/隐藏按钮的控制，则使用这个参数， 那么它和{% include href/button/columnsToggle.button %}不同的地方在于单个列和多个列操作

这个参数可以根据class名称、jquery选择器或者是列索引来指定到列的配置， 你也可以通过{% include href/option/buttons.buttons.text %}配置按钮上显示的文字

选项(Options)

这个按钮有以下选项配置:

名称 默认
text 取决于你定义的列标题
默认情况下，按钮上显示的文字就是列的标题，你还可以通过 {% include href/option/buttons.buttons.text %}配置按钮上显示的文字
className buttons-columnVisibility
按钮的class名称，详情参见 {% include href/option/buttons.buttons.className %}
action
设置列的可见性
columns undefined
Columns selector that defines the columns to include in the column visibility button set. By default this is {% include href/Types.html param="undefined" %} which results in all columns being selected, but any of the {% include href/Types.html param="column-selector" %} options can be used to define a custom button set.
visibility undefined
用来给选中的列设置值，true为显示， false为隐藏， undefined为切换当前状态

示例

Datatables初始化：只显示第二列的控制按钮 

$('#myTable').DataTable( {
    buttons: [ {
        extend: 'columnToggle',
        columns: 1
    } ]
} );

Datatables初始化：显示两个按钮，通过class名称 primary和 secondary来指定列 

$('#myTable').DataTable( {
    buttons: [
        {
            extend: 'columnToggle',
            columns: '.primary'
        },
        {
            extend: 'columnToggle',
            columns: '.secondary'
        }
    ]
} );

直接初始化：使用jquery选择器来指定列 

new $.fn.dataTable.Buttons( table, {
    buttons: [
        {
            extend: 'columnToggle',
            columns: ':gt(4)'
        }
    ]
} );
