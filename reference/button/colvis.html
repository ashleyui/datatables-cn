colvis

起始版本号: {{ page.since }}

一个按钮集合提供列可见性控制

注意：这个需要使用Bottons扩展

说明(Description)

将创建一组按钮，来切换列的可见性

选项(Options)

这个按钮有以下选项配置:

名称 默认
text Column visibility
用文字显示，使用 {% include href/option/buttons.buttons.text %} 配置显示的文字内容 或者 buttons.colvis,关于语言的配置参考 {% include href/option/Internationalisation.html param="language" %}
className buttons-collection buttons-colvis
按钮的class名称，详情参见 {% include href/option/buttons.buttons.className %}
action
显示一个集合来控制列的可见性
columns undefined
Columns selector that defines the columns to include in the column visibility button set. By default this is {% include href/Types.html param="undefined" %} which results in all columns being selected, but any of the {% include href/Types.html param="column-selector" %} options can be used to define a custom button set.

示例

Datatables初始化：使用默认选项显示 colvis 按钮 

$('#myTable').DataTable( {
    buttons: [ 'colvis' ]
} );

直接初始化：显示colvis 按钮，但是不包含第一列 

new $.fn.dataTable.Buttons( table, {
    buttons: [
        {
            extend: 'colvis',
            columns: ':gt(0)'
        }
    ]
} );
