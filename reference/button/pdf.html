pdf

起始版本号: {{ page.since }}

以pdf格式保存表格中的数据

注意：这个需要使用Bottons扩展

说明(Description)

这个按钮作为一个代理，根据浏览器的支持和页面上的库自动选择是 {% include href/button/pdfHtml5.button %}还是 {% include href/button/pdfFlash.button %}

如果浏览器不支持，按钮则不会显示

选择的顺序是：

  • {% include href/button/pdfHtml5.button %}
  • {% include href/button/pdfFlash.button %}

如果检测到环境要求不满足，则不显示此按钮。

Datatables已经处理了该使用html5还是falsh，这样不需要程序员用逻辑去控制， 或者你设置 message来选择Html5或者Flash按钮类型。 例外的是，customize选项仅用于Html5按钮。 如果是Flash 生成的PDF，它将被忽略掉。

选项(Options)

这个按钮有以下选项配置:

名称 默认
text PDF
按钮上显示的文字，使用 {% include href/option/buttons.buttons.text %} 配置显示的文字内容 或者 buttons.copy,关于语言的配置参考 {% include href/option/Internationalisation.html param="language" %}
className buttons-pdf
按钮的class名称，详情参见 {% include href/option/buttons.buttons.className %}
action
创建并保存一个PDF文件
exportOptions {}
选择性的导出数据，包括哪些列，行，排序或者过滤。 详细参考 {% include href/api/Buttons.html param="buttons().exportData()" %} 方法，这里收集所需要的数据
header true
表头将包含在导出的数据中
footer false
表脚将包含在导出的数据中
title *
给创建文件的文件名，*代表替换成系统默认的标题标签。
extension .pdf
给创建文件的文件名后缀。
orientation portrait
纸张的方向设定。 portraitString纵向， landscapeString横向
pageSize A4
纸张的大小设定。 A3String， A4String， LEGALString（216*356）， LETTERString（216*279）， TABLOIDString（279*432） （美国国家标准化组织 ANSI 定义了自己的标准 X3.151-1987）
message
可选的描述信息，显示在创建pdf的上面。
customize null

A function that can be used to customise the PDFMake document definition object for the {% include href/button/pdfHtml5.button %} button. The document object is passed in as the only parameter to this function and should be manipulates as required. A return value is not expected or used.

Please refer to the PDFMake documentation for information on how to manipulate this object and the options available for the generated PDF.

示例

Datatables初始化：使用 {% include href/button/pdf.button %}，让浏览器自动选择 

$('#myTable').DataTable( {
    buttons: [
        'pdf'
    ]
} );

Datatables初始化：使用 exportOptions 选项配置导当前页数据 

$('#myTable').DataTable( {
    buttons: [
        {
            extend: 'pdf',
            text: 'Save current page',
            exportOptions: {
                modifier: {
                    page: 'current'
                }
            }
        }
    ]
} );
