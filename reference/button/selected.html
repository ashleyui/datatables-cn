一个或者多个项目被选中时需要开启

注意：这个需要使用 Select扩展的支持

说明(Description)

当用户在选择表中的项目时，你可能希望提供用户一个动作，这个按钮提供了一个基本的 框架，动态的启用和禁用自动选择表中的项目。

通过一个 action的 function提供此功能， 参考 则这个按钮会被激活，参考 {% include href/option/buttons.buttons.action %}/{% include href/api/Buttons.html param="button().action()" %}

示例

Datatables初始化：计算多少行被选中了 

$('#myTable').DataTable( {
    buttons: [
        {
            extend: 'selected',
            text: 'Count selected rows'
            action: function ( e, dt, button, config ) {
                alert( dt.rows( { selected: true } ).indexes().length +' row(s) selected' );
            }
        }
    }
} );
