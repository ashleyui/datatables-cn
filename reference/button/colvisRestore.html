colvisRestore

起始版本号: {{ page.since }}

恢复列可见性原来的状态

注意：这个需要使用Bottons扩展

说明(Description)

用来恢复到之前的可见状态

选项(Options)

这个按钮有以下选项配置:

名称 默认
text CSV
用文字显示，使用 {% include href/option/buttons.buttons.text %} 配置显示的文字内容 或者 buttons.colvisRestore, 关于语言的配置参考 {% include href/Options.html param="language" %}
className buttons-colvisRestore
按钮的class名称，详情参见 {% include href/option/buttons.buttons.className %}
action
重置列的可见性

示例

Datatables初始化：显示列的可见性和恢复按钮 

$('#myTable').DataTable( {
    buttons: [
        'colvis',
        'colvisRestore'
    ]
} );
