ajax.url().load()

起始版本号: 1.10

设置新的url并加载数据

说明

使用{% include href/api/Core.html param="ajax.url()" %}方法来设置新的请求地址， load()方法才会有效

数据类型

functionajax.url().load( callback, resetPaging )

参数:
名称 类型 是否可选
1 callback {% include href/function.js %} Yes - default:null

当服务器返回数据并重绘完毕时执行此回调方法，回调方法返回的是服务器返回的数据

2 resetPaging {% include href/boolean.js %} Yes - default:true

重置（默认或者设置为true）或者保持分页信息（设置为false）

返回:

{% include href/Types.html param="DataTables.Api" %},

DataTables.Api 实例

示例

首先用data.json的数据初始化表格，然后立即用newData.json加载数据 

var table = $('#example').DataTable( {
    ajax: "data.json"
} );

table.ajax.url( 'newData.json' ).load();
