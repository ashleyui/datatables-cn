page()

起始版本号: {{ page.since }}

获取/设置表格的当前页

说明

分页是Datatables的一个核心特性，该方法提供了手动的控制页面的显示。如果你想自定义分页控件，参考 这里

注意：如果你视图操作一个不存在的页数，Datatables不会抛出错误，而是重置到第一页

还需要注意的是，使用 page()时需要搭配 {% include href/api/Core.html param="draw()" %} 方法一起使用，比如 table.page( 0 ).draw( false );

另外，这里需要加 false在 draw()方法里，没有这个参数，表格会做一个完全的重绘，并且回到第一页

数据类型

functionpage()

描述:
获取表格当前页。注意：如果当前api实例里有多个表格对象，那么返回的是第一个对象的当前页，你应该使用 {% include href/api/Tables.html param="table()" %}来确保操作的是单个表格对象
返回:

{% include href/integer.js %}

当前页数

functionpage( set )

描述:
设置表格显示第几页
参数：
名称 类型 是否可选
1 set {% include href/integer.js %}，{% include href/string.js %} No

参数有以下两种方式:

  • integer - 你要跳转到的页面索引 (0代表第一页)
  • string - 代表一个操作，dt定义了下面的字符串:
    • first - 跳到第一页
    • next - 下一页
    • previous - 上一页
    • last - 最后一页
返回:

{% include href/Types.html param="DataTables.Api" %},

DataTables.Api 实例

示例

自定义控制页面的操作 

var table = $('#example').DataTable();

$('#next').on( 'click', function () {
    table.page( 'next' ).draw( false );
} );

$('#previous').on( 'click', function () {
    table.page( 'previous' ).draw( false );
} );
