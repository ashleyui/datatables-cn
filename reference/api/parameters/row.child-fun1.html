名称（Name） 类型（Type） 是否可选（Optional）
1 showRemove {% include href/Types.html param="boolean" %} No
此参数可以指定为true或false：
  • true，附加到父级的任何子级行将立即变为可见。这相当于使用{% include href/APIs.html param="row().child.show()" %}
  • false，如果父行当前有任何子行（无论显示与否），此方法将销毁那些子行，并在适当的情况下将其从DOM中删除。与{% include href/APIs.html param="row().child.hide()" %}方法不同，此选项从Datatables保留的内存数据中完全删除行。这相当于使用{% include href/APIs.html param="row().child.remove()" %}。
