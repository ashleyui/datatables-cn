从数据源读取新的数据。

默认情况下，DataTables 会自动给行 从以前最初被读取过的数据源（比如，dom或者js数据源）中读取新数据。 但是这个参数可以用来重载以及明确地告诉DataTables应该在哪个新数据里读取。这个可能在你使用的是AJAX数据，但是又使用JS去修改单元格DOM内容的时候很有用。

这个属性可以接受如下几个参数：
  • {% include href/string.html param="auto" %} - 使用原始的数据源
  • {% include href/string.html param="data" %} - 使用JavaScript数据结构
  • {% include href/string.html param="dom" %} - 使用当前保存在DOM中的数据
