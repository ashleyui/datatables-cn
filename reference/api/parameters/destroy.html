1 remove {% include href/type/Javascript.html param="boolean" %} Yes-default:false
  • true - 完全从DOM中删除表
  • false - 保留DOM，恢复到原始的未增强的HTML状态(默认false)

在v1.10.8版本中，如果使用 true ，DataTables 将会使用 jQuery .remove()方法从页面中删除表，这导致 绑定到表元素的任何事件被jQuery自动移除。如果设置 false 自定义事件不会从表中删除，DataTables本身附加的事件会自动移除。
