1 paging {% include href/type/Javascript.html param="boolean" %},{% include href/type/Javascript.html param="string" %} Yes - default:true
此参数决定什么样的重绘DataTables将执行。这里有三个选项可以提供（请注意，字符串选项需要v1.10.8或者更新的版本）
  • {% include href/string.html param="full-reset" %}或者 true(默认) - 排序和搜索将会重新计算，行在新的位置重新绘制。 分页将回到第一页。
  • {% include href/string.html param="full-hold" %}或者 false - 排序和搜索将会重新计算，行在新的位置重新绘制。 分页不会被重置，任保持当前页。
  • {% include href/string.html param="page" %} - 排序，搜索和分页都维持当前的。这个会很有用，在页数（即{% include href/api/Core.html param="page()" %} ） 切换之间数据不会改变。
请注意，字符串选项需要v1.10.8或者更新的版本，v1.10.8版本以下只支持 boolean 类型的值
