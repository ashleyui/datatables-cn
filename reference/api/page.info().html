page.info()

起始版本号: {{ page.since }}

获取表格的分页信息

说明

分页信息告诉用户当前的数据信息，第几页或者总共多少条数据等等。 当{% include href/Options.html param="paging" %}是禁用时，你可以通过 page.info()来获取分页信息

该方法提供表的分页状态，表中的记录数（总记录数和过滤后的记录数）

返回的对象有以下属性：

  • page - 当前页的索引（0代表第一页）
  • pages - 总页数
  • start - 起始索引
  • end - 结束索引
  • length - 每页长度。一般情况下 start+length=end，但是也不是所有情况都是这样，例如你总共数据条数是2，而 length是10
  • recordsTotal - 总条数
  • recordsDisplay - 显示的数据条数，如果你加上了过滤条件这个则是过滤后的记录数

数据类型

functionpage.info()

描述:
获取表格分页状态。注意：如果当前api实例里有多个表格对象，那么返回的是第一个对象的当前页，你应该使用 {% include href/APIs.html param="table()" %}来确保操作的是单个表格对象
返回:

{% include href/object.js %}

表格的分页信息， 返回对象的样例： 

{
    "page": 1,
    "pages": 6,
    "start": 10,
    "end": 20,
    "length": 10,
    "recordsTotal": 57,
    "recordsDisplay": 57
}

示例

获得当前页面和表的记录数: 

var table = $('#example').DataTable();
var info = table.page.info();

$('#tableInfo').html(
    'Currently showing page '+(info.page+1)+' of '+info.pages+' pages.'
);
