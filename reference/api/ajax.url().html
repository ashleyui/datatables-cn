ajax.url()

起始版本号: 1.10

设置新的url数据源

说明

你可以使用{% include href/api/Core.html param="ajax.reload()" %}方法来刷新表格数据，但是在某些情况下你需要重新设置其他的数据源 这个时候那个使用这个方法会帮到你

注意，使用 {% include href/api/Core.html param="ajax.url()" %} 方法只能设置新的url，并不能触发ajax请求，你需要使用 {% include href/api/Core.html param="ajax.url().load()" %} 方法来跟换url并重新获取数据

数据类型

functionajax.url()

返回:

{% include href/string.js %},

注意，如果是多个实例，那么返回的是设置了ajax的第一个table实例的url，或者使用{% include href/api/Tables.html param="table()" %}先指定实例后再操作

functionajax.url( url )

参数:
名称 类型 是否可选
1 url {% include href/string.js %} NO

给表格设置新的url数据源

返回:

{% include href/type/DataTables.html param="DataTables.Api" %},

DataTables.Api 实例

示例

得到当前url 

var table = $('#example').DataTable( {
    ajax: 'data.json'
} );

alert( 'Data source: '+table.ajax.url() ); // 将打印 'Data source: data.json'

设置新的数据源，并立即加载 

var table = $('#example').DataTable( {
    ajax: "data.json"
} );

table.ajax.url( 'newData.json' ).load();
