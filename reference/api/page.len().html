page.len()

起始版本号: {{ page.since }}

获取/设置表格的分页长度

说明

这个方法仅仅就是设置/获取分页的长度，需要{% include href/Options.html param="paging" %}是打开的

一个特别的数字 -1可以作为参数，代表是所有的数据

同样这个需要配合 {% include href/api/Core.html param="draw()" %} 方法才能应用到表格上。比如 table.page( 0 ).draw();

数据类型

functionpage.len()

描述:
获取表格分页长度。注意：如果当前api实例里有多个表格对象，那么返回的是第一个对象的当前页，你应该使用 {% include href/api/Tables.html param="table()" %}来确保操作的是单个表格对象
返回:

{% include href/integer.js %}

当前页长度，如果是 -1表示所有数据

functionpage.len( set )

描述:
获取表格分页状态。注意：如果当前api实例里有多个表格对象，那么返回的是第一个对象的当前页，你应该使用 {% include href/api/Tables.html param="table()" %}来确保操作的是单个表格对象
参数：
名称 类型 是否可选
1 set {% include href/integer.js %} No

设置分页长度. 使用 -1 代表显示所有数据.
返回:

{% include href/Types.html param="DataTables.Api" %},

DataTables.Api 实例

示例

用户自定义显示分页长度，是显示10条还是显示所有 

var table = $('#example').DataTable();

$('#all').on( 'click', function () {
    table.page.len( -1 ).draw();
} );

$('#_10').on( 'click', function () {
    table.page.len( 10 ).draw();
} );
