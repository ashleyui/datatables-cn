名称 类型 描述
DT_RowId {% include href/Types.html param="string" %} 将{% include href/HtmlTags.html param="tr" %}节点的ID属性设置为此值
DT_RowClass {% include href/Types.html param="string" %} 添加这个样式到{% include href/HtmlTags.html param="tr" %}节点上
DT_RowData {% include href/Types.html param="object" %} 使用 jQuery data()方法把对象数据绑定到行中，方便之后用来取出（比如在一个点击事件）
DT_RowAttr {% include href/Types.html param="object" %} 将对象中包含的数据作为属性添加到{% include href/HtmlTags.html param="tr" %}节点。 对象的键作为属性的键，对象的值作为属性的值。这个是使用 jQuery param()方法。请注意这个选项 需要 Datatables 1.10.5或者更新的版本才支持。
